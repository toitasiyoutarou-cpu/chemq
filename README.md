# CHEM QUEST｜実験まとめ 二連戦

## そのままGitHub Pagesで使う方法
1. Zipを解凍する
2. 中身（index.html、imgフォルダ、README.md）をGitHubリポジトリにアップロード
3. Settings → Pages → Deploy from branch → main / root を選択

## 内容
- 第1戦：熱分解スライム
  - 炭酸水素ナトリウムの熱分解の式を、質量から判断する
  - 最終問題：結論として適切な化学反応式を選ぶ
- 第2戦：CaCO3ゴーレム
  - 未知濃度塩酸の考察
  - 最終問題：20mL塩酸のモル濃度を計算する

## 仕様
- 正解位置は毎回ランダム
- 不正解なら同じ問題に再挑戦
- 解説は正誤に関わらず表示
- 最終問題は「かいしんのいちげき！」
